# तत्वो का अर्थ — Tatva Ka Gyan

A digital library for Jain scriptures, bhajans, poojas, and religious texts.

## 🌐 Live Website

`https://[your-username].github.io/tatva-ka-gyan/`

## 🚀 Quick Deploy to GitHub Pages

### 1. Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/[your-username]/tatva-ka-gyan.git
git push -u origin main
```

### 2. Enable GitHub Pages
1. Go to repository **Settings** → **Pages**
2. Under "Build and deployment", select **GitHub Actions**
3. Done! Site deploys automatically

## 💻 Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
